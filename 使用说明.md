# 链动小铺商家增强工具

QQ 交流群：`1076144676`

脚本文件：`ldxp-merchant-toolkit.user.js`

## 安装

1. 浏览器安装 Tampermonkey（篡改猴）或 Violentmonkey。
2. 打开脚本管理器，选择“添加新脚本”。
3. 删除编辑器内原内容，粘贴 `ldxp-merchant-toolkit.user.js` 的全部内容并保存。
4. 登录链动小铺商家后台，刷新目标页面。

## 货源广场

打开：`https://www.ldxp.cn/merchant/my_parent/source_square`

页面右下角会出现“增强货源搜索”：

- 可按关键词、类型、价格、库存、对接状态筛选。
- 每个搜索结果后面有“一键对接”。
- 支持勾选多个结果后批量对接。
- 默认按成本价加价 10%，可在面板中修改加价比例。

## 商品管理

打开：`https://www.ldxp.cn/merchant/goods/list?is_proxy=1`

页面右下角会出现“商品批量工具”：

1. 选择商品类型、状态和拉取页数，点击“加载商品”。
2. 可先选择“筛选分类”，只加载该分类下的商品。
3. 可使用“库存筛选”查看全部、有库存或缺货商品；非卡密商品显示为“无需库存”。
4. 可使用“价格排序”按价格从低到高或从高到低展示。
5. 勾选需要处理的商品；“分类”列会按分类 ID 自动显示正确名称。
6. 在“搜索分类选项”输入名称，选择“修改为分类”，再批量保存。
7. 还可批量修改售价或上下架状态。
8. 点击“生成并复制文字报表”，生成以下格式：

```text
商品名称：示例商品
库存：100
价格：¥19.90
链接：https://...
```

## 安全说明

- 脚本只请求 `ldxp.cn` 自己的 `/merchantApi/` 接口，不向第三方发送数据。
- 批量修改和对接前都会显示确认框。
- 建议先勾选 1 个测试商品验证分类、价格和链接结果，再进行大批量操作。
